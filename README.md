# 📊 Superstore Sales Analysis & Forecasting

## 🚀 Overview
This project analyzes a retail sales dataset to uncover business insights and predict future sales trends. It combines **data cleaning, exploratory data analysis (EDA), and time series forecasting** to support data-driven decision-making.

---

## 🎯 Objectives
- Identify trends and patterns in sales performance  
- Analyze customer segments, cities, and shipping modes  
- Discover seasonal sales behavior  
- Forecast future sales using time series models  

---

## 🛠️ Tech Stack
- Python  
- Pandas
- Matplotlib, Seaborn  
- Statsmodels (ARIMA)  
- Prophet  

---

## 📂 Dataset
- Superstore Sales Dataset (`train.csv`)  
- Source: https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting

---

## 📊 Exploratory Data Analysis (EDA)
- 📈 Sales trend over time  
- 🏙️ Top-performing cities  
- 👥 Customer segment analysis  
- 🚚 Shipping mode performance  
- 📅 Monthly and seasonal patterns  

---

## 🤖 Forecasting

### 🔹 ARIMA Model
- Used for statistical time series forecasting  
- Captures trends in historical sales data  

### 🔹 Prophet Model
- Handles trend and seasonality (daily, weekly, yearly)  
- Provides interpretable forecasting components  

---

## 📈 Key Insights
- Sales show clear seasonal patterns with peak periods  
- Certain cities and customer segments drive most revenue  
- Shipping modes influence sales distribution  
- Forecasting models indicate continued trends with fluctuations
